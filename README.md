# School Management System

A comprehensive school management website with role-based access control for admins, teachers, and students.

## Features

- **Admin Dashboard**: Manage teachers, students, and view analytics
- **Teacher Dashboard**: Manage classes, students, and approve registrations
- **Student Dashboard**: View courses, progress, and grades
- **Security System**: Role-based access control with approval workflow
- **Progress Tracking**: Monitor student performance and progress

## Usage

Visit the website and login with your credentials:
- Admin accounts can manage the entire system
- Teacher accounts can manage their classes and approve student registrations
- Student accounts need to be approved before accessing the dashboard
